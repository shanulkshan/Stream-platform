# Stream-platform

The Front-end is created with React vite.

The Back-end is created with Node.js

command To start frontend -> "npm run dev"

command to start backend -> "npm start"

---------------------------------------------------

Here, the first thing to navigate to is the interface corresponding to the 1.jpeg provided in the resources. By clicking on the cards there, you can navigate to the interface corresponding to the 4.jpeg image.

However, if the user is not signed-in, they will have to sign in before navigating to another page. By clicking on the cards in the 4.jpeg interface, you can navigate to the 3.jpeg.

---------------------------------------------
Tailwind CSS is used for styles.

jwt tokens are used for user authentication.
